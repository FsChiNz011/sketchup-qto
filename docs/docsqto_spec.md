\# QTO Specification



\## Input



Supported:



\- Group

\- Component



\## Model Type



Solid Models



\## Output



Quantity Report



Columns:



\- Entity

\- Category

\- Quantity

\- Unit



\## Example



Wall A | Wall | 24.5 | m²



Wall B | Wall | 18.2 | m²



Column C1 | Column | 0.85 | m³
